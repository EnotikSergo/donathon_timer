В обс можно делать захват через приложение, а так же через браузер вписав ссылку localhost:41701/overlay/, если нужно к примеру виджет для стрима с телефона, для этого нужен хост, я лично пользовался программой cloudpub чтобы дать странице публичный адрес:

Настройка для оверлея в PRISM(картинки в папке)
1. Зарегистрироваться и скачать программу(для того чтобы к таймеру можно было подключиться не только из локальной сети) https://cloudpub.ru

2. Добавить адрес: Все оставить как есть только порт поменять на 41701

3. Публичный адрес может быть большим, можно изменить поддомен для удобства

4. Далее скопировать публичный адрес и в веб источнике PRISM ввести https://test.cloudpub.ru/overlay/ ну или для теста ты можешь просто в браузере его ввести должен открыться таймер

Кнопки:

- "R" - сбросить значение
- Arrow UP - добавить минуту
- Shift + Arrow UP - добавить час
- Arrow DOWN - убрать минуту
- Shift + Arrow DOWN - убрать час
- Space - пауза/продолжить
- "O" - настройки
